# 90s Retro Space Shooter

A top-down 90s style arcade space shooter built with HTML5 Canvas, Vanilla Javascript, and CSS.

## Features
- Classic neon 90s retro aesthetic
- Smooth top-down player movement
- Endless enemies to shoot down
- Score tracking and Game Over states
- Clean, object-oriented code structure

## How to Run
1. Open this folder in your terminal or file explorer.
2. Serve the directory using a local web server (e.g., `npx http-server` or `python -m http.server`).
3. Alternatively, depending on your browser's security settings, you can simply open `index.html` in your web browser.

## Controls
- **Arrow Keys** or **W, A, S, D**: Move the player ship.
- **Spacebar**: Shoot bullets.
