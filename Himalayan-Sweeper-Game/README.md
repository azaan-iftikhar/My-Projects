# Himalayan Sweeper

A 90s-style Minesweeper game themed around mining pink salt. Features a classic Windows 95 UI aesthetic.

## Features
- Windows 95 styling (beveled borders, system fonts)
- Pink salt theme with emoji icons
- Recursive empty-space revealing
- Digital timer and mine counter

## How to play
1. Open `index.html` in your web browser.
2. Left-click a cell to reveal it.
3. Right-click a cell to place a flag on a suspected mine.
4. Clear the board without clicking any mines!
