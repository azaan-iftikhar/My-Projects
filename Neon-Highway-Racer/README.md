# Neon Highway Racer

A pseudo-3D endless highway racing game built with HTML5 Canvas and Vanilla JS.

## Features
- **Pseudo-3D Rendering**: Perspective projection of road lines and scenery.
- **Traffic & Collectibles**: Dodge varying-speed traffic and collect coins.
- **Nitro Boost**: Use the spacebar for a sudden speed burst, complete with screen shake and FOV warp.
- **Mobile Friendly**: Touch steering (tap left/right side of screen) and device orientation (tilt) support.
- **Synth Audio**: Web Audio API generated engine hum and sound effects.
- **Persistence**: High score saved locally.

## Controls
- **PC**: `W/S` or `Up/Down` to Accelerate/Brake. `A/D` or `Left/Right` to steer. `Space` for Nitro.
- **Mobile**: Tap left/right side of the screen to steer. Accelerates automatically.

## How to Run
Just open `index.html` in any modern web browser. No server required.
