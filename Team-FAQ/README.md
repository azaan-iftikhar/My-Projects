# Team FAQ

A clean, responsive, and easily searchable Team FAQ webpage built with HTML, CSS, and JavaScript.

## Features
- Displays 5 common workplace questions and answers
- Search bar to filter questions dynamically in real-time
- Accordion style for expandable and collapsible answers
- Fully responsive design for both desktop and mobile
- Clean UI utilizing a professional Teal and Charcoal color scheme

## How to Run
1. Clone or download this directory to your local machine.
2. Double-click the `index.html` file to open it in your default web browser. No server setup is required.

## Project Structure
- `index.html`: Contains the core structure of the page, the search input, and the FAQ container.
- `style.css`: Contains the styling and responsive design rules.
- `script.js`: Contains the logic for the accordion behavior and real-time filtering.
