# Invoice Generator

A professional, client-side invoice generator built with HTML, CSS, and JavaScript.

## Features
- **Dynamic Items Table**: Add, remove, and auto-calculate line items.
- **Red/White Theme**: Clean, responsive design optimized for both desktop and mobile.
- **Real-time Calculations**: Subtotal, Tax, and Grand Total update instantly as you type.
- **Draft Saving**: Save your progress to the browser's local storage and resume later.
- **Print Optimization**: Dedicated print styling strips away form elements and buttons for a clean physical or PDF invoice.
- **Form Validation**: Ensures all required fields are filled before previewing or printing.

## How to Run
Since this is a client-side application with no build steps or backend:
1. Navigate to the folder containing these files.
2. Open `index.html` directly in any modern web browser (Chrome, Firefox, Safari, Edge).

## Structure
- `index.html`: The semantic layout and user interface.
- `styles.css`: The red/white themed styling, responsiveness, and print overrides.
- `script.js`: State management, event handling, calculations, and local storage integration.
